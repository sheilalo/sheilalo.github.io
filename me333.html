PID Motor Control January 2017 - March 2017
Mechatronics

Overview

As part of ME 333: Introduction to Mechatronics, I created a PID motor control system in which a user could specify angles and trajectories for the control system to follow. The motor control code for the PIC32 was written in C, which then interfaced with the client user interface in MATLAB.

Process

In order to control the position of the motor, I used two nested PID loops. The inner loop fed back on the current to the motor while the outer loop fed back on position, as seen below.

I created a text-based user interface in MATLAB to command the PIC over a UART connection. With the proper input, the user could tell the motor to move to a position, follow a trajectory, etc. In some cases while tuning the controller, I discovered a "propeller" mode due to the poor gains.
